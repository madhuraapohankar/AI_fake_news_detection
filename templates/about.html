  • Trained model prediction

🔐 Authentication

  • Live & Video news
  • File upload

🛠 Technologies

  • Flask (Python)
  • Scikit-learn
  • SQLite
  • HTML, CSS, JS

🎯 Purpose

Demonstrates AI integration with full-stack development in a secure and scalable web system.
